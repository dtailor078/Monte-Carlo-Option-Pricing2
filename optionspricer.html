Key concepts
Spot price (S₀) input
The current market price of the underlying asset — what it's trading at right now.
Strike price (K) input
The price at which you have the right to buy (call) or sell (put) the asset when the option expires.
Call vs put input
A call profits if the price rises above the strike. A put profits if the price falls below the strike.
Volatility (σ) input
How much the asset price fluctuates annually. Higher vol = wider range of outcomes = more expensive option.
Risk-free rate input
The return on a riskless investment (e.g. government bonds). Used to discount future payoffs to today's value.
Time to maturity input
How long until the option expires, in years. More time = more uncertainty = higher option value.
MC price output
The option price estimated by averaging discounted payoffs across thousands of simulated price paths.
Black-Scholes price output
The analytical closed-form price. Used as a benchmark — MC should converge to this with enough paths.
95% CI output
Confidence interval on the MC estimate (±1.96 × std error). Narrows as simulation count increases.
Delta output
How much the option price changes for a $1 move in the underlying. Ranges 0–1 for calls, -1–0 for puts.
Intrinsic value output
The payoff if exercised right now — max(S−K, 0) for a call. Option price is always ≥ intrinsic value.
Geometric Brownian motion model
The stochastic process used to simulate price paths: drift (risk-free rate) plus random Gaussian shocks.
